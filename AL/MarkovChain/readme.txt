What is MarkovChain Game?
******************
MarkovChain is a sample app TonyLi design and implement for demonstrate my Java skills



How to start:
***************
On Windows : 
	run cmd and switch to MarkovChain folder, then run "startup.bat"
on Linux:
	sh startup.sh ( you can type "chmod 777 startup.sh" first, if needed)

how to build:
***********
If you want to build the project yourself, make sure you have installed ANT and set JAVA_HOME correctly.
Then, run cmd, switch to RecipeFinder folder, type "ant". 
You can get the package in ./classes/markov-chain.zip
